Navigation Shortcuts

Speed up your workflow with keyboard shortcuts

Speed Up Your Workflow

Master keyboard shortcuts and navigation techniques to move through Sophia quickly and efficiently.

Global Shortcuts

  • Ctrl+N - Create new snippet
  • Ctrl+S - Save current snippet
  • Ctrl+F - Search snippets
  • Ctrl+/ - Toggle comments
  • Ctrl+Shift+S - Save as new snippet
  • Escape - Close modal/dialog

Navigation Shortcuts

  • Ctrl+1-8 - Quick access to main sections
  • Alt+Left/Right - Navigate back/forward
  • Ctrl+Tab - Next tab/window
  • Ctrl+Shift+Tab - Previous tab/window

Editor Shortcuts

  • Ctrl+D - Select next occurrence
  • Ctrl+Shift+L - Select all occurrences
  • Alt+Up/Down - Move line up/down
  • Ctrl+Shift+K - Delete current line
  • Ctrl+Shift+Enter - Insert line below

Workflow Optimization

  • Use the command palette (Ctrl+Shift+P)
  • Create custom shortcuts in settings
  • Use split view for multiple snippets
  • Enable vim/emacs mode for power users
