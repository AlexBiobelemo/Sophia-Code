Master the art of finding what you need quickly

Master the Art of Searching

Learn advanced search techniques to find exactly what you need quickly and efficiently.

Basic Search

  • Simple search: Type keywords to search titles and content
  • Tag search: Use tag:python to find Python snippets
  • Language filter: Select from the language dropdown

Advanced Search Operators

  • title:"function" - Search in titles only
  • content:"array" - Search in code content
  • tag:python,javascript - Multiple tags
  • author:username - Find snippets by specific user
  • created:2024 - Filter by creation year
  • language:python AND tag:web - Combine filters

Smart Search Features

  • Fuzzy matching: Find similar words and typos
  • Auto-complete: Suggestions as you type
  • Recent searches: Access your search history
  • Saved searches: Bookmark frequently used queries

Search Tips

  • Use quotes for exact phrases: "function declaration"
  • Combine operators for precise results
  • Use wildcards: func* finds "function", "functional", etc.
  • Filter by date ranges for recent or old snippets
