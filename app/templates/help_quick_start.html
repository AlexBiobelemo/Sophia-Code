Getting Started with Sophia

Welcome to Sophia! This guide will help you get up and running in just a few minutes.

Step 1: Create Your First Snippet

  • Click on the "Create Snippet" button in the main navigation
  • Enter a title for your code snippet
  • Select the appropriate programming language
  • Paste or type your code in the editor
  • Add relevant tags and descriptions
  • Click "Save" to store your snippet

Step 2: Organize Your Snippets

  • Create collections to group related snippets
  • Use tags to categorize your code
  • Set visibility levels (private, public, or shared)

Step 3: Start Searching

  • Use the search bar to find snippets by title, content, or tags
  • Apply filters for language, type, or date
  • Bookmark frequently used snippets

Quick Tips

  • Use Ctrl+S to quickly save snippets
  • Press Ctrl+/ to toggle comments
  • Use the AI assistant for code generation and optimization
  • Export snippets in various formats for sharing
