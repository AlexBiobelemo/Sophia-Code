## Introduction

Welcome to Project Sophia! This comprehensive guide will walk you through using all features of the application to manage your code snippets, solve programming problems, and leverage AI for code generation.

## Account Setup

### Registration

1. Navigate to the **Register** page by clicking "Register" in the navigation bar.
2. Enter a unique **Username** (must be at least 4 characters).
3. Enter a valid **Email** address.
4. Enter a **Password** (at least 8 characters recommended).
5. Confirm your password in the **Repeat Password** field.
6. Click the **Register** button.

### Login

1. Navigate to the **Login** page by clicking "Login" in the navigation bar.
2. Enter your **Username** and **Password**.
3. (Optional) Check **Remember Me** to stay logged in across browser sessions.
4. Click the **Sign In** button.

> **Note:** After 5 failed login attempts, your account will be locked for 30 minutes for security.

### Profile Management

1. Click on your username or profile icon in the navigation bar to access your **User Profile**.
2. Click **Edit Profile** to change:
   - **Username** - Your display name
   - **Email** - Your contact email
   - **Avatar** - Upload a profile picture (PNG, JPG, GIF, or WebP)
   - **Current Password** - Required to make changes
   - **New Password** - Leave blank to keep current password

### User Settings

Access User Settings from your profile or navigation:

- **AI Settings:**
  - Preferred AI Model: Choose between Gemini models or Minimax
  - Code Generation Style: Balanced, Detailed, or Concise
  - Auto Explain Code: Automatically explain code on creation

- **UI Preferences:**
  - Show Line Numbers: Display line numbers in code snippets
  - Enable Animations: Enable/disable UI animations
  - Enable Tooltips: Show helpful tooltips on hover
  - Tooltip Delay: Set delay before tooltips appear (seconds)
  - Dark Mode: Toggle dark/light theme

- **Notifications:**
  - Email Notifications: Receive email updates

- **Privacy:**
  - Auto Save Snippets: Automatically save snippet drafts
  - Public Profile: Make profile visible to others
  - Show Activity: Display activity on profile

## Managing Code Snippets

### Creating a New Snippet

1. Click **Create Snippet** in the navigation bar.
2. Fill in the form:
   - **Title** - A descriptive name for your snippet
   - **Collection** - (Optional) Select a collection to organize your snippet
   - **Language** - Select the programming language for syntax highlighting
   - **Description** - (Optional) Explain what your code does
   - **Code** - Paste or write your code
   - **Tags** - (Optional) Comma-separated tags for categorization
3. Click **Save Snippet**.

### Viewing Snippets

1. On the homepage, click any snippet card to view details.
2. Use the **Previous** and **Next** buttons to navigate between snippets.
3. Click **Edit** to modify the snippet.
4. Click **Delete** to remove the snippet (confirmation required).

### Editing Snippets

1. View the snippet you want to edit.
2. Click the **Edit** button.
3. Modify any fields as needed.
4. Click **Save Changes**.
5. A version snapshot is automatically created before saving.

### Snippet Version History

1. View a snippet.
2. Click **History** to see all versions.
3. View changes between versions.
4. Click **Revert** to restore a previous version.

### Deleting Snippets

1. View the snippet.
2. Click **Delete**.
3. Confirm the deletion in the dialog.

> **Warning:** Deleting a snippet cannot be undone. Consider exporting first.

## Managing Collections

### Creating Collections

1. Navigate to **Collections** in the navigation bar.
2. Click **Create New Collection**.
3. Enter a **Collection Name**.
4. (Optional) Select a **Parent Collection** for nested organization.
5. Click **Create Collection**.

### Viewing Collections

1. Go to the **Collections** page.
2. Click on a collection to view its contents.
3. View snippet count and nested sub-collections.

### Renaming Collections

1. Go to the **Collections** page.
2. Click the **Rename** button next to a collection.
3. Enter the new name and optionally change the parent collection.
4. Click **Save Changes**.

### Deleting Collections

1. Go to the **Collections** page.
2. Click the **Delete** button next to a collection.
3. Confirm deletion.

> **Warning:** Deleting a collection does not delete its snippets. Snippets become uncollected.

### Reordering Collections

1. Go to the **Collections** page.
2. Use the **up** (↑) and **down** (↓) arrow buttons to reorder collections.
3. Or use drag-and-drop if available.

## Moving and Copying Snippets

### Move a Snippet

1. View the snippet.
2. Click **Move**.
3. Select the target collection.
4. Click **Move**.

### Copy a Snippet

1. View the snippet.
2. Click **Copy**.
3. Select the target collection.
4. Click **Copy**.
5. A new copy is created with "Copy of" prefix.

### Bulk Operations

1. On the homepage or collection view, select multiple snippets using checkboxes.
2. Use the **Bulk Actions** dropdown:
   - **Delete** - Remove selected snippets
   - **Copy** - Copy to a collection
   - **Move** - Move to a collection
3. Click **Perform Bulk Action**.

## AI-Powered Features

### Code Generation

1. Navigate to **AI Code Generation** or click **Generate**.
2. Enter a description of the code you want in the prompt field.
3. Click **Generate Code**.
4. Review the generated code and explanation.
5. Click **Create Snippet** to save it, or **Retry** to regenerate.

### Multi-Step Thinking Generation

1. Navigate to **AI Code Generation**.
2. Enter your problem description.
3. (Optional) Provide test cases for verification.
4. The AI will run through 4 layers:
   - **Architecture** - Problem analysis and strategy
   - **Coder** - Code generation
   - **Tester** - Verification and debugging
   - **Refiner** - Optimization and final review
5. Review all stages and the final code.

### AI Code Explanation

1. View a snippet.
2. Click **Explain with AI**.
3. View the detailed explanation including:
   - Overview and intent
   - How it works
   - Key design decisions
   - Complexity analysis
   - Edge cases and correctness
   - Improvements and alternatives

### AI Code Formatting

1. View a snippet in edit mode.
2. Click **Format with AI**.
3. The AI will format the code with proper indentation and style.
4. Review and save the formatted code.

### AI Tag Suggestion

1. When creating or editing a snippet, click **Suggest Tags**.
2. The AI analyzes your code and suggests relevant tags.
3. Accept or modify the suggestions.

### AI Code Refinement

1. When code has errors, click **Refine with AI**.
2. Enter the error message or output.
3. The AI diagnoses and provides corrected code.

### AI Note Explanation

1. View a note.
2. Click **Explain** to get an AI-powered explanation of the note content.

## LeetCode Problem Management

### Adding a Problem

1. Navigate to **Add Problem**.
2. Enter the **Problem Title**.
3. Enter the **Problem Description**.
4. Select the **Difficulty** (Easy, Medium, or Hard).
5. Add **Tags** (comma-separated) for categorization.
6. (Optional) Provide the **LeetCode URL** if it's from LeetCode.
7. Click **Add Problem**.

### Generating Solutions

1. View a problem.
2. Click **Generate Solution**.
3. Select the **Solution Language**.
4. Click **Generate Solution**.
5. Review the generated solution and explanation.
6. Click **Approve Solution** if it meets requirements.

### Approving Solutions

1. Navigate to the solution (from problem page or notifications).
2. Review the solution code and explanation.
3. Click **Approve** to publish the solution.
4. Approved solutions are visible in search results.

### Viewing Solutions

1. View a problem.
2. See all approved solutions in the Solutions section.
3. Click a solution to view details.

## Search

### Basic Search

1. Use the search bar in the navigation or on the homepage.
2. Enter keywords to search across snippets.
3. Press Enter or click Search.

### Advanced Search Syntax

Use these operators for precise searches:

- `tag:python` - Search by tag
- `lang:javascript` - Search by language
- `collection:Algorithms` - Search in collection
- `in:title,code` - Search in specific fields
- `before:2024-01-01` - Created before date
- `after:2024-01-01` - Created after date
- `-deprecated` - Exclude results with "deprecated"
- `"binary search"` - Exact phrase match

### Semantic Search

The system automatically performs semantic search using AI embeddings, finding conceptually related snippets even without exact keyword matches.

### Solution Search

1. Navigate to **Solutions Search**.
2. Use the same search syntax to find approved solutions.
3. Results include problem context and explanations.

## FAQ

### How do I reset my password?

Currently, password reset requires database access. Contact support if needed.

### Can I import snippets from other sources?

Import functionality is not yet implemented. You can manually create snippets or use the export/import file format.

### What languages are supported?

All major programming languages are supported for syntax highlighting including:
Python, JavaScript, Java, C, C++, C#, Go, Ruby, PHP, Swift, Kotlin, Rust, TypeScript, SQL, HTML, CSS, and more.

### How does AI pricing work?

The application uses your own API keys. You are charged by Google Gemini or Minimax based on your API usage. Monitor your API console for usage and costs.

### How do I delete my account?

Account deletion is not available through the UI. Contact support for account removal.

### Can I share snippets with others?

Snippet visibility is currently private. Public profile setting allows others to see your badges and statistics but not your snippets.

### How do I report a bug?

Check the Troubleshooting section below. If the issue persists, check the GitHub repository issues or contact support.

### What happens to my data?

Your data is stored in the SQLite database (`app.db`). Regular backups are recommended. Enable auto-snapshots for daily Markdown backups.

### How do I enable dark mode?

Go to **User Settings** → **UI Preferences** → Toggle **Dark Mode**.

### Can I use Project Sophia offline?

No, AI features require an internet connection. Local snippet management works offline.

## Tips and Best Practices

1. **Use Tags Effectively:** Add relevant tags to make snippets searchable.
2. **Organize with Collections:** Use nested collections for large projects.
3. **Write Descriptions:** Add clear descriptions to explain snippet purpose.
4. **Use Version History:** Don't worry about experimenting—history allows reverting.
5. **Leverage AI:** Use AI for code explanation and formatting to learn best practices.
6. **Export Regularly:** Backup your snippets using the export feature.
7. **Maintain Streaks:** Daily activity builds streaks and earns badges.
8. **Use the Chat:** Ask the AI assistant for help with code problems.
